# Results Directory

This folder contains two kinds of artifacts:

1. Preserved historical artifacts already present in the repository before the refactor.
2. Structured outputs generated by the new training and evaluation scripts.

## Currently committed files

- `metrics.json`: seeded with preserved historical metrics from the original README plus notes about the exported notebook images
- `model_comparison.csv`: historical comparison table from the pre-refactor README
- `classification_report.txt`: reports derived from the preserved confusion matrix PNG exports
- `confusion_matrix.png`: preserved Random Forest confusion matrix export
- `feature_importance.png`: preserved Random Forest feature importance export
- `logistic_regression_confusion_matrix.png`: preserved Logistic Regression confusion matrix export
- `sample_predictions/example_input.json`: documented example input payload for inference

## Important provenance note

The currently tracked notebook file does not contain embedded XGBoost training cells or structured metric outputs. As a result:

- the XGBoost metrics in the committed `metrics.json` and `model_comparison.csv` are preserved from the original repository README,
- the Random Forest and Logistic Regression confusion matrices come from the PNG exports that were already tracked in the repository.

Running `scripts/run_train.py` and `scripts/run_evaluate.py` will overwrite `metrics.json`, `model_comparison.csv`, `classification_report.txt`, `confusion_matrix.png`, and `feature_importance.png` with fresh script-generated outputs.
